Internet of Things

“Oh, so they have internet on computers now!” ~Homer Simpson

The Internet of Things (often abbreviated as "IoT") refers to ever-growing network of physical objects that feature an IP address and can communicate using the Internet.

Consumer Examples: Weather stations, garage doors, baby monitors, Canary security system, Hue light bulbs, WeMo plugs, Nest thermostat, Amazon Echo ...
